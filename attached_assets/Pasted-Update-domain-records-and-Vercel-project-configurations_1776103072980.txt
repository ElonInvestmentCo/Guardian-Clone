how do i point the main site to Vercel In Zoho?

3 hours ago
In Zoho, you point your main site to Vercel by editing your domain’s DNS records.

1. Add the domain in Vercel first
In Vercel:

Open your project.
Go to Settings → Domains.
Add:
guardiiantrading.com
www.guardiiantrading.com

Vercel will show the DNS records it expects.

2. Open Zoho DNS Manager
In Zoho:

Go to your domain manager.
Open the DNS settings for:
guardiiantrading.com

Look for existing website records for:
@
www
Remove old website records that point to Render, Netlify, Railway, Replit, etc.

Do not remove Zoho email records like MX, SPF, DKIM, or DMARC.

3. Add Vercel records in Zoho
For the root domain:

Type: A
Host / Name: @
Value / Points to: 76.76.21.21
TTL: Automatic or 3600

For www:

Type: CNAME
Host / Name: www
Value / Points to: cname.vercel-dns.com
TTL: Automatic or 3600

4. Keep API pointed to Render
Since you use Render for the API, add or keep this record too:

Type: CNAME
Host / Name: api
Value / Points to: guardian-trading-api.onrender.com
TTL: Automatic or 3600

Then in Render, make sure you add:

api.guardiiantrading.com

as a custom domain.

5. Set the API URL in Vercel
In Vercel:

Go to Project → Settings → Environment Variables.
Add:
VITE_API_URL=https://api.guardiiantrading.com

Redeploy the Vercel project.
Final result
Your website:

https://guardiiantrading.com
https://www.guardiiantrading.com

Your API:

https://api.guardiiantrading.com/api/healthz

DNS can take a few minutes to a few hours to fully update.
